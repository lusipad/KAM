### 1.2 分层架构详细说明

#### 表示层 (Presentation Layer)
| 组件 | 技术选型 | 职责 |
|------|----------|------|
| Web应用 | React + TypeScript | 主用户界面 |
| 桌面端 | Electron/Tauri | 本地优先体验 |
| 移动端 | React Native | 移动设备支持 |
| 插件系统 | VS Code Extension API | IDE集成 |

#### 应用层 (Application Layer)
| 服务 | 职责 | 核心功能 |
|------|------|----------|
| 知识管理服务 | 文档生命周期管理 | CRUD、版本控制、协作编辑 |
| 记忆管理服务 | 长期记忆维护 | 存储、检索、压缩、遗忘 |
| ClawTeam引擎 | 多代理协调 | 任务分配、执行监控、结果聚合 |
| 项目同步服务 | 外部系统集成 | 数据映射、同步策略、冲突解决 |

#### 领域层 (Domain Layer)
- 核心业务逻辑封装
- 领域事件定义
- 聚合根设计
- 值对象定义

#### 基础设施层 (Infrastructure Layer)
- 技术实现细节
- 外部服务适配器
- 数据访问实现
- 消息队列实现
